# Podcast Search Rows

## Fallback Titles

PodcastSearchRows uses 'Episode' and 'Transcript' as fallback titles. <!-- [^0f3f2-58] -->

Tapping a transcript search result seeks the player to that timestamp and opens the episode. <!-- [^rollo-16] -->
