# Podcast Sidebar Navigation

## Sidebar Navigation

The podcast sidebar displays a single nav row that opens the podcast list view, not an inline list of podcasts. Agent-generated podcasts intentionally do not create a PodcastSubscription row, causing them to appear in 'See All' but not in the user's followed-podcasts list.
