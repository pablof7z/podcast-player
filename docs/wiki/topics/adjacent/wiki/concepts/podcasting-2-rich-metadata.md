# Podcasting 2 Rich Metadata

Podcasting 2.0 is a strong adjacent layer because it extends the open RSS ecosystem with metadata that maps directly to rich product surfaces.

## Product Ideas

- **Cloud chapter navigation**: show external chapter files with images, summaries, topic chips, and agent-generated gaps when publisher chapters are missing.
- **Soundbite gallery**: render publisher soundbites and AI-detected highlights as shareable clips.
- **People graph**: use person tags as seeds for host, guest, contributor, and recurring-speaker profiles.
- **Creator recommendations**: use podrolls as a high-trust discovery channel.
- **Location lens**: map episodes by editorial subject or production location.
- **Live listening room**: support liveItem streams, reminders, chat, and post-live episode conversion.
- **Alternate media picker**: expose video, audio-only, high-quality, low-bandwidth, language, or commentary-track options from alternateEnclosure.

## Why It Fits Podcastr

Podcastr already needs transcripts, people, chapters, semantic search, and generated clips. Podcasting 2.0 metadata can become a trusted publisher-provided input layer before the agent falls back to inference.
