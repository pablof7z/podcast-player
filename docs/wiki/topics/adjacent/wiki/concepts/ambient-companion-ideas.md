# Ambient Companion Ideas

Podcasts are often consumed away from the full app. Adjacent richness should therefore appear in ambient surfaces, not only inside a beautiful tab view.

## Product Ideas

- **Action Button voice mode**: one physical gesture opens "talk to my podcasts."
- **Lock Screen controls**: resume briefing, save snip, ask about current moment, skip to next chapter.
- **Live Activity with citations**: current episode, chapter, transcript line, and agent-suggested related moment.
- **Widget stack**: today's briefing, transcript-ready queue, unfinished threads, and "ask about this week."
- **Control Center controls**: start briefing, save timestamp, sleep timer, toggle voice mode.
- **Generated video overview**: for visual learners, create a simple narrated slide-style summary from a podcast cluster.
- **Live episode support**: liveItem can power live reminders, live listening, and post-live archive processing.

## Design Rule

Ambient features should perform one clear action. The full app can handle exploration; system surfaces should capture, resume, ask, or play.
